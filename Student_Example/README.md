# Student Project Submissions

This repository is used to collect and display student projects.

## Instructions for Students

1. Fork this repository to your GitHub account
2. Open your forked repository
3. Create a folder with your full name (example: Student_Ali_Khan)
4. Add your project files inside your folder
5. Do NOT change or delete other studentsâ€™ folders
6. Create a Pull Request to submit your work

## Important Rules

- One folder per student
- Folder name must contain your name
- Late submissions may not be accepted
